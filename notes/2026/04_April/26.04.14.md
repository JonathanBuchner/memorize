### SortedDictionary<TKey,TValue> already enumerates in ascending key order
SortedDictionary<TKey,TValue> already enumerates in ascending key order, so iteration begins at the smallest key automatically. A normal foreach starts from the minimum and moves upward.
```csharp
foreach (var pair in dict)
{
    Console.WriteLine($"{pair.Key} : {pair.Value}");
}
```
